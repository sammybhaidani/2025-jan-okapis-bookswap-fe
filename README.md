# BookSwap
[Back-end](https://github.com/sammybhaidani/book-swap-api) |
[Live demo](https://okapibookswap.netlify.app/)

A React-based book swap application that allows users to browse, claim, return, and review books from a shared library.

## Tech Stack

React, React router, Tailwind CSS

## Features

- View available and claimed books with cover images and details
- Filter the book collection by genre categories
- View comprehensive information including title, author, year, page count, and blurb
- Claim available books by providing your name
- Return books you've claimed back to the available pool
- Read and write reviews with star ratings for books
- Contribute new books to the library with full details and cover images

## Getting Started

### Installation

To run this project locally, follow these steps:

1. **Clone the repository**

```sh
git clone git@github.com:sammybhaidani/2025-jan-okapis-bookswap-fe.git
```

2. **Navigate to the project directory:**

```sh
cd 2025-jan-okapis-bookswap-fe
```

3. **Install dependencies:**

```sh
npm install
```

4. **Start the development server:**

```sh
npm run dev
```

5. **Access the application:**

Open your browser and navigate to the local development URL
(typically `http://localhost:5173`)
